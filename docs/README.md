# Documentation AuditAX 🚀

Bienvenue dans la documentation officielle d'**AuditAX**, la plateforme de conformité intelligente industrialisée.

## 📁 Structure du Projet
- `/src` : Source du frontend React/Vite.
- `/src/server` : Backend Express.js & Prisma.
- `/docs` : Documentation utilisateur et technique.
- `openapi.yaml` : Contrat d'interface de l'API.

## 🛠️ Guide de Développement
1. **Installation** : `npm install`
2. **Base de données** : `npx prisma db push` (Puis `npm run seed`)
3. **Lancement** : `npm run dev` (Lance le serveur Express et le frontend Vite en parallèle).
